🤝 Contribute

Help expand and improve these open datasets

How to Contribute

1
Fork the repository
Go to github.com/juanmoisesd and fork the relevant dataset repository.
2
Add or correct data
Edit the CSV/Excel file or add new data following the variable format in CODEBOOK.md.
3
Submit a Pull Request
Submit a PR with a clear description of what changed and why.
4
Review process
All contributions are reviewed within 7 days. Accepted contributions are credited in CHANGELOG.md.
5
Alternative: Open an Issue
If you find an error or want to suggest data without a PR, open a GitHub Issue.
